# QZIJKPlayer
基于 IJK的简易视频播放器，支持切换多个播放源、全屏、跟随屏幕旋转全屏

![](https://tva1.sinaimg.cn/large/007S8ZIlgy1gflcrqszvaj31ds0n0jyo.jpg)



功能比较简单，UI布局主要使用的 xib，可以自己在上面添加所需功能，自动横屏旋转需要系统打开自动旋转功能。

IJKMediaFramework可以自行编译，也可以使用我的编译的
